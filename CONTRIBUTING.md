# Come contribuire

Grazie per l'interesse verso la comunità del Fontanin! Questo progetto è
aperto sia a chi vuole contribuire tecnicamente sia a chi vuole partecipare
in altri modi (manutenzione area, comunicazione, eventi).

## Non tecnici

- Proponi idee o segnalazioni aprendo una issue su GitHub, oppure tramite i
  canali social della community (vedi README).
- Per partecipare alla manutenzione o agli eventi, contatta un referente
  d'area (vedi [docs/GOVERNANCE.md](./docs/GOVERNANCE.md)).

## Tecnici

1. Fai fork o crea un branch dal `main`.
2. `npm install` e `npm run dev` per lavorare in locale.
3. Segui lo stile del codice esistente (componenti in `src/components`,
   pagine in `src/pages`).
4. Apri una Pull Request descrivendo la modifica e collegando l'eventuale
   issue di riferimento.
5. Non committare mai `.env.local` o credenziali: sono già esclusi da
   `.gitignore`.

## Processo decisionale

1. Proposta (issue).
2. Discussione nel canale/incontro periodico della community.
3. Approvazione da parte del referente d'area (o assemblea per decisioni
   rilevanti).
4. Esecuzione tracciata tramite issue/PR.
